# Potion Shop 🧪  
A small full‑stack practice project built with Java + Springboot for the backend and HTML, CSS, and JavaScript for the frontend. The database uses PostgreSQL.
The goal is to create a retro‑styled potion shop interface with a working backend and a containerized database.

This project is still **under construction** as features, UI layout, and backend routes are still being developed.  
More details, screenshots, and setup instructions will be added as I continue working on the project.
